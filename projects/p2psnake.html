P2P Snake Game

A peer-to-peer multiplayer Snake game featuring real-time networking and competitive gameplay between players.

Role Developer
Type Multiplayer Game
Stack P2P Networking

Overview

A modern take on the classic Snake game, this project implements peer-to-peer networking so two players can compete in real-time without a central server. Players race to grow their snake while avoiding collision with each other and the boundaries.

Key Features

  • Peer-to-peer multiplayer without a central server
  • Real-time game state synchronization between players
  • Competitive multiplayer with collision detection
  • Smooth rendering and responsive controls
