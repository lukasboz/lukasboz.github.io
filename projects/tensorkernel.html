TensorKernel

A high-performance C++ tensor computation library featuring SIMD vectorization, OpenMP multi-threading, and CUDA GPU acceleration, with seamless Python/PyTorch interoperability via pybind11.

Role Developer
Type Systems Library
Stack C++, CUDA, SIMD, OpenMP, Python, PyTorch, pybind11

Overview

TensorKernel is a from-scratch tensor computation library built in C++ for maximum performance. It leverages SIMD intrinsics for vectorized CPU operations, OpenMP for multi-threaded parallelism, and CUDA kernels for GPU acceleration -- providing a flexible backend for numerical computing.

Key Features

  • SIMD-optimized tensor operations for fast CPU computation
  • OpenMP multi-threading for parallel workloads
  • CUDA kernel integration for GPU-accelerated computation
  • Python bindings via pybind11 for seamless PyTorch interop
  • Custom memory management optimized for tensor operations
  • Benchmarked against standard libraries for performance validation
